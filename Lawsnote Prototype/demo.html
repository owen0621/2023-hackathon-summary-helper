高等法院 107,重勞上,68 損害賠償等 民事 判決
上訴人: 亞太雷射工業股份有限公司   /   被上訴人: 黃敬益
   對於爭議的客戶名單是否屬於營業秘密，法院認為需要根據營業秘密法的要件來判斷。如果客戶名單只包含一般的資訊且容易取得，且沒有其他重要的客戶資訊，則難以認定其具有營業秘密的要件。然而，如果被上訴人在未經授權的情況下使用或洩漏了該客戶名單，則可認定為違約或不法使用營業秘密。根據證人的證述和相關證據，法院認為被上訴人違反了營業秘密法，並使用了上訴人的營業秘密，即系爭客戶名單，侵害了上訴人的營業秘密權益。法院判決被上訴人應賠償上訴人80萬3,192元，並支付法定遲延利息。其他請求部分不予准許。
高等法院 105,勞上,53 損害賠償 民事 判決
上訴人: 高柏科技有限公司   /   被上訴人: 李維鋐、蔡建宏、饒瑞鈞、王苡倫
    蔡建宏、饒瑞鈞和王苡倫與上訴人所簽署的僱傭契約中，有關於競業禁止的約定是有效的。然而，關於禁止從事與公司相同或類似性質的工作以及禁止成為同類事業無限責任之股東的約定則被認為是無效的，因為這些約定沒有提供合理的補償。蔡建宏和饒瑞鈞在任職期間已經與他人共同設立了與上訴人相同業務的公司，並利用職務機會進行不公平競爭行為，違反了競業禁止約定和忠誠義務，應該承擔違約賠償責任。饒瑞鈞在與友達公司的交易中提供不實資訊，導致上訴人損失，因此需連帶賠償上訴人450萬元及利息。蔡建宏和李維鋐也需連帶賠償上訴人。王苡倫未參與不法行為，不予賠償。艾美公司及李維鋐需支付利息直到清償日止。
高等法院 臺中分院 106,上易,581 損害賠償 民事 判決
上訴人: 德樂仕企業有限公司   /   被上訴人: 盧佑林
    上訴人主張被上訴人違反公司法及民法之規定，應負損害賠償責任。然而，法院認為上訴人未能證明被上訴人為其公司經理人，且被上訴人在離職後並未違反競業禁止規定。此外，上訴人主張被上訴人為公司實質監察人，應負有忠實執行業務及盡善良管理人之注意義務，但法院認為此主張亦無據。因此，法院判決駁回上訴，維持原審判決。
高等法院 臺南分院 106,上更(一),4 民事
上訴人: 立山企業股份有限公司   /   被上訴人: 陳辰雄
    法院認為上訴人公司無法證明被上訴人違反忠誠執行職務及善良管理人義務的主張。法院指出，上訴人公司的經營困境與內部矛盾有關，並非被上訴人的責任。法院認為，被上訴人陳辰雄在成立立固公司後的行為並不涉及原上訴人公司的訂單流失或協力廠商的移轉，上訴人未能提供足夠的證據來證明被上訴人陳辰雄將客戶訂單、營業秘密和協力廠商等事項無償移轉或洩露給立固公司，也未能證明訂單轉移到立固公司。此外，法院認為被上訴人陳良政成立立固公司並不違反公司法的規定，上訴人未能提供證據來證明被上訴人陳良政有意違法召開股東會資遣員工與停業。最後，法院認為上訴人未能提供足夠的證據來證明被上訴人的行為導致上訴人公司的損失，因此不應准許上訴人的賠償請求。
高等法院 臺南分院 106,上易,26 請求回復營業 民事 判決
上訴人: 立山企業股份有限公司   /   被上訴人: 陳辰雄、陳良政
    被上訴人乙的決議雖然違反了程序，但在實質上是否違反了忠實執行業務和善良管理人的注意義務，尚有疑問。法院指出，上訴人公司因內部家族間的爭執已達到難以溝通協調的地步，且黃氏家族對被上訴人提起多件損害賠償訴訟，另外還有刑事告訴，顯示兩造之間的糾紛已經非常嚴重，難以繼續經營公司。此外，上訴人公司所接訂單過去大部分是委由辰豐鐵工廠代工，而辰豐鐵工廠在102年6月30日終止代工關係後，上訴人公司無法找到其他協力廠商代工，導致金屬製品的營業額減少，無法應付各項支出，使公司陷入虧損狀態。最後，法院指出上訴人提出的證據資料並無法證明辰豐鐵工廠非上訴人公司的協力廠商，且證人的證詞也無法支持上訴人的主張。因此，法院認為上訴人的主張不足以成立。根據以上判決，法院認為上訴人未能提出足夠的證據來支持其主張，因此判決不支持上訴人的請求。
高等法院 106,上,327 損害賠償 民事 判決
上訴人: 漢唐光電科技股份有限公司   /   被上訴人: 黃柏強
    本案為上訴人對於原審判決不服提起上訴，要求被上訴人賠償損害。然而，法院認為系爭競業條款無效，且被上訴人在離職後的競業行為並未違反競業禁止條款。因此，法院判決駁回上訴，並要求上訴人負擔訴訟費用。法院認為，系爭競業條款不符合公序良俗，且未提供合理的補償措施，因此無效。此外，法院也認為上訴人未能證明被上訴人在離職後的競業行為對其造成實際損害，因此不予賠償。綜上所述，法院認為上訴人的上訴理由不成立，判決駁回上訴。
高等法院 103,重勞上,21 侵權行為損害賠償等 民事 判決
上訴人: [???]   /   被上訴人: [???]
    陳男在擔任泰盛興公司的經理人期間，涉嫌違反了經理人的忠實義務和競業禁止義務，並且進行了不完全給付和侵權行為。陳男以席克斯公司的名義進行了一系列的交易，賺取了價差，導致泰盛興公司損失了大量的利益。陳男的行為還涉嫌背信罪。 泰盛興公司要求陳男賠償損失的要求是有據的。 法院判決陳男賠償相應的金額，並加計法定遲延利息。 泰盛興公司的損害賠償請求並未罹於時效，因為泰盛興公司在合理的時間內才得知陳男的侵權行為。
