高等法院 107,重勞上,68 損害賠償等 民事 判決

1. 當事人或當事公司：
     - 上訴人：在台灣從事雷射板金加工業的公司，上訴人名稱為益昌公司。
     - 被上訴人：曾在上訴人公司工作，後來創立自己的公司，被上訴人名稱為東雷多公司。
2. 法官認定之事實：
     - 被上訴人在離開上訴人公司後，創立了自己的公司（東雷多公司），並從上訴人公司引進了客戶。
     - 被上訴人使用了上訴人公司的客戶名單，並在其新公司（益昌公司）繼續與這些客戶交易。
3. 法官判斷該事實是否該當法律構成要件：
     - 法官認為被上訴人的行為構成了侵害上訴人公司的營業秘密，因為他使用了客戶名單和營業資訊。
     - 法官指出營業秘密法第12條第1項規定，被上訴人有侵害營業秘密的行為。
4. 法院判斷及法院判斷的理由：
     - 法院認為被上訴人應負賠償責任，根據營業秘密法第12條第1項，法院判決被上訴人應支付給上訴人80萬3,192元。
     - 法院還判斷被上訴人應支付遲延利息，自106年10月14日起按年息5%計算。
5. 請求權基礎以及根據之法律條文：
     - 上訴人的請求權基礎是營業秘密法第12條，根據這個法律條文，上訴人要求被上訴人支付賠償金。
     - 法官根據營業秘密法判決被上訴人支付80萬3,192元的賠償金，以及遲延利息。
高等法院 105,勞上,53 損害賠償 民事 判決

1. 兩造當事人或當事公司具體身分及姓名：
     - 原告：高柏公司
     - 被告：艾美公司、李維鋐、蔡建宏、饒瑞鈞
2. 法官認定之事實：
     - 原告聲稱被告侵害其商譽，並要求損害賠償。
     - 原告主張被告違反競業禁止約定，造成客戶流失損害。
     - 原告聲稱被告以不實電郵損及伊公司商譽。
3. 法官判斷該事實是否該當法律構成要件：
     - 法院認為原告未能證明被告的行為構成侵權行為，並拒絕了其損害賠償請求。
     - 法院認定被告確實參與了競爭，導致原告的客戶流失。
     - 法院認為原告未能證明被告的電郵損害其商譽。
4. 法院判斷及法院判斷的理由：
     - 在所有案件中，法院一致認為原告未能提供足夠的證據證明被告的行為對其商譽造成實際損害。
5. 請求權基礎以及根據之法律條文：
     - 原告基於侵權行為主張損害賠償，參照相關民事訴訟法相關條文提出訴訟。
高等法院 臺中分院 106,上易,581 損害賠償 民事 判決

1. 兩造當事人或當事公司具體身分及姓名：
     - 上訴人：德樂仕企業有限公司（法定代表人：劉維平）
     - 被上訴人：盧佑林
2. 法官裁定的事實：
     - 被上訴人並非上訴人公司依法委任的經理人或股份有限公司的監察人。
     - 被上訴人不具備公司經理人或股份有限公司監察人的資格。
     - 上訴人未能提供充分證據證明被上訴人實際上具備公司經理人的資格。
     - 有限公司的不經營業務股東只能進行質詢公司經營情況、查閱財產文件、帳簿、表冊等行為，不具備代表公司行使權利的權限。
3. 法官判定是否構成法律要件：
     - 法官裁定被上訴人未違反相關法規，因為其不具備公司經理人的身份，也未違反經理人競業禁止規定。
4. 法院判決及判決理由：
     - 法院拒絕了上訴人的主張，認為上訴人未能證明被上訴人實際上具備公司經理人的資格，且被上訴人不違反相關法規。因此，法院判決上訴被拒絕。
5. 請求權基礎以及根據之法律條文：
     - 根據法官的判決，上訴人主張的基礎是涉及公司法第32條和民法第562條等相關法律規定，但法院認為這些規定並不適用於被上訴人的情況。法官未具體引用法律條文，但根據摘要中提到的相關法規，可以理解法官的判決基礎。
高等法院 臺南分院 106,上更(一),4 民事

1. 兩造當事人或當事公司：
     - 上訴人：蔡有明
     - 被上訴人：盧定墉
     - 第三人：陳建志
2. 法官認定之事實：
     - 系爭契約約定上訴人出售房地給被上訴人，價金分期付款，其中第3期款遲延支付。
     - 被上訴人主張系爭土地有被他人占用，上訴人未履行排除占用的責任。
     - 第三人陳建志為系爭土地的實際占有人。
3. 法官判斷：
     - 法官認定被上訴人在簽訂契約時已知悉系爭土地被他人占用，但兩造並未就按現況點交達成共識。
     - 法官判斷上訴人未按期支付第3期款，但被上訴人的解除契約行為未生效。
     - 法官認定第三人陳建志為系爭土地的實際占有人，但不認同其主張的排除占用權。
4. 法院判斷及理由：
     - 原判決撤銷了原先關於超過部分的判斷，並駁回被上訴人的訴求。
     - 法院認為上訴人的解除契約行為不合法，被上訴人有權主張繼續履行契約。
     - 對於第三人的排除占用主張，法院未予支持。
5. 請求權基礎以及根據之法律條文：
     - 上訴人蔡有明主張系爭契約失效，並援引民法第190條。
     - 被上訴人盧定墉主張繼續履行契約，並援引民法第195條。
高等法院 臺南分院 106,上易,26 請求回復營業 民事 判決

1. 當事人：
     - 上訴人: 立山企業股份有限公司
     - 被上訴人: 陳辰雄、陳良政
2. 法官認定之事實：
     - 上訴人主張被上訴人在公司事務上違反忠實執行職務及善良管理人的義務，並進行不法侵權行為。
     - 上訴人聲稱被上訴人的行為導致上訴人公司受損害，包括未能繼續營業所需的費用。
3. 法官判斷是否構成法律要件：
     - 法官認為，上訴人未能充分證明被上訴人的行為違反忠實執行職務及善良管理人的義務，且未能證明行為與損害之間具有相當的因果關係。
     - 法官指出，上訴人提出的營業所需費用中包括公司的常規開支，並未能充分證明與被上訴人的行為直接相關。
4. 法院判斷及理由：
     - 法院判決上訴人的上訴無理由，並駁回其追加之訴。
     - 法院認為上訴人未能充分證明其主張的侵權行為，且未能確定與公司損失之間的因果關係。
     - 根據公司法第23條第1項、民法第544條、第185條第1項的規定，法院拒絕了上訴人的請求。
5. 請求權基礎及相關法條：
     - 上訴人根據公司法第23條第1項、民法第544條、第185條第1項的規定，請求被上訴人連帶給付100萬元，以及自起訴狀繕本送達翌日起至清償日止，按週年利率5%計算之利息。
高等法院 106,上,327 損害賠償 民事 判決

1. 兩造當事人或當事公司：
     - 上訴人：正達國際光電股份有限公司
     - 被上訴人：許銘栗
2. 法官認定之事實：
     - 上訴人持有公司債，其中包括正達二公司債。
3. 法官判斷該事實是否該當法律構成要件：
     - 法官認定上訴人持有正達二公司債。
4. 法院判斷及法院判斷的理由：
     - 法院判斷上訴人有清償被上訴人許銘栗所持有的正達二公司債或提供足額擔保的義務。
     - 法院參考公司法第281條、民法第295條、系爭調解結果等相關法條和文件，認為上訴人應履行清償或提供擔保的義務。
5. 請求權基礎以及根據之法律條文：
     - 上訴人根據公司法第281條、民法第295條、系爭調解結果等法條，請求被上訴人許銘栗清償其持有的正達二公司債或提供足額擔保。
高等法院 103,重勞上,21 侵權行為損害賠償等 民事 判決

1. 兩造當事人或當事公司具體身分及姓名：
     - 泰盛興公司（以下簡稱泰盛興）
     - 席克斯公司
     - 陳堃樹
     - 弘盛興公司
     - 邱能老
2. 法官認定之事實：
     - 泰盛興向席克斯公司出售氧化鋁產品。
     - 陳堃樹曾是席克斯公司的實際負責人。
     - 陳堃樹違反公司法第32條的競業禁止義務。
     - 泰盛興主張陳堃樹及席克斯公司對其的侵權行為造成經濟損害。
3. 法官判斷該事實是否該當法律構成要件：
     - 法院認定陳堃樹違反了公司法第32條的競業禁止義務，並導致泰盛興的經濟損害。
4. 法院判斷及法院判斷的理由：
     - 泰盛興主張陳堃樹及席克斯公司應對其的經濟損害負連帶賠償責任。
     - 法院根據民法第184條和第188條的相關規定，認定陳堃樹對泰盛興公司的損害負連帶賠償責任。
5. 請求權基礎以及根據之法律條文：
     - 泰盛興公司主張連帶賠償責任的基礎是民法第184條和第188條。
